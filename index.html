Important Concepts

The internet, the web, browsers and HTML

The internet is a way that allows computers (phones, tablets, servers etc) around the world to talk to one another. We send these information packets (bytes) through LAN cables or through WiFi. It used to be just text. Now computers talk using a new language called HTML (Hypertext Markup Language) to create a web page. So instead of just text, HTML allows the webpage to be in a graphical format. We will need a browsers to view HTML in a graphical format, which you will know it as a webpage. All these web pages is collectively known as the world wide web or commonly known as the web. So the internet is the physical highway between computers, the vehicles on the highway are the packets of information. There are many different types of vehicles now. The car is like the web traffic, the trucks like the email traffic and the motorcycles like your music, video, picture traffic. So the internet contains the the web, email, music, video, pictures etc.

HTML terms

HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web.

Tags and Elements

HTML documents are made of HTML elements. When writing HTML, we tell browsers the type of each element by using HTML tags. Eg tag = bold

Why Computers are Stupid

Computers are stupid because they interpret instructions literally. Every tag has to be correct for the computer to understand what you want it to do.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

So how does everthing work together?

When you enter the url for a webpage like www.google.com, your computer sends a HTTP Request to a server using HTML. The server finds the appropriate HTML document and sends it back to your computer. Your web browser on your computer will interpret the HTML into the webpage that you ultimately see on your computer screen.
